1 | filter()

The filter() function creates a subset of the data based on the value of one or more columns. Use filter() to answer the questions below.


Do you remember we had that ‚ÄúAll‚Äù category in the Star Wars scrap data? Let‚Äôs take a look at that.

filter(starwars_scrap, Origin == "All")

Pro-tip!

We use a == (double equals sign) for comparing values. A == makes the comparison ‚Äúis it equal to?‚Äù and returns a True or False answer. So the code above returns all the rows where the condition Origin == "All" is TRUE.

A single equals sign (=) is used within functions to set options, for example read_csv(file = "starwars_scrap_jakku.csv"). Don‚Äôt worry too much. If you use the wrong symbol R is often helpful and will let you know which one is needed.
